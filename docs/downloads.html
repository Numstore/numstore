Installation Methods

From Source

The recommended way to install Numstore is to build from source:

git clone https://gitlab.com/lincketheo/numstore.git
cd numstore
make
sudo make install

Python Package

Install the Python wrapper via pip:

# First, install Numstore C library
# Then install Python wrapper
pip install numstore

Note: Python package requires Numstore C library to be installed first.

Java Package

Maven dependency (coming soon):

<dependency>

System Requirements

Minimum Requirements

  • OS: Linux 3.10+, macOS 10.12+, Windows 10+
  • Memory: 128 MB RAM
  • Disk: 50 MB for installation
  • Compiler: GCC 7+, Clang 8+, or MSVC 2019+
  • Build Tools: CMake 3.10+

Recommended Requirements

  • Memory: 1 GB+ RAM for better performance
  • Disk: SSD for optimal I/O performance
  • CPU: Multi-core processor for parallel operations

Verify Installation

After installation, verify Numstore is working:

# Check version
./nstore --version

# Run tests
./build/debug/apps/test

# Try a simple operation
./nstore "create test i32"
echo -e "\\x01\\x00\\x00\\x00" | ./nstore "insert test 0"
./nstore "read test 0:1:1" | hexdump -C
./nstore "delete test"

Development Builds

Get the latest development version:

cd numstore
git checkout main
make

Warning: Development builds may be unstable. Use release versions for production.
