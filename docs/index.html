What is Numstore?

Numstore is a database for contiguous bytes. It's essentially a file system, but in database format with log N mutations in the middle.

Numstore uses an "R+Tree" - a rope equivalent to a B+Tree that keeps track of the count of bytes each node stores. It's designed for efficient operations on byte streams with logarithmic time complexity for insertions, deletions, and reads anywhere in the stream.

Use Cases

Numstore is ideal for:

  • Scientific data storage with efficient random access
  • Time-series databases requiring fast insertions and reads
  • Machine learning datasets with complex access patterns
  • Engineering simulations requiring large contiguous data structures
  • Applications needing log N insertions/deletions in byte streams
